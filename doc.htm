jquery.ejb.slidepanel.v1.5.js

Avalible options (with default settings):
pause: 5, // this is the time the slide waits before it loads the next slide, decimals are permited.
start: 0, // this sets the starting slide
autoPlay:true, // tells the slide if it should automaticly progress the slides
loop:true, // when we reach the last slide should we start over?
playBtn:null, // pass in the class or id of the DOM object/objects you want to control the play status of this slidepanel
pauseBtn:null, // pass in the class or id of the DOM object/objects you want to control the pause status of this slidepanel
nextBtn:null, // pass in the class or id of the DOM object/objects you want to control the loading of the next slide
prevBtn:null, // pass in the class or id of the DOM object/objects you want to control the loading of the previous slide
slideButtons:null, // pass in the class or id of the DOM object/objects you want to recieve a series of divs that control loading individual slides
slideButtonsAutoCreate:true, // should we be looking for anything to populate with slideButtons?
slideButtonsClass:"button", // the class used to show a slideButton
slideButtonsSelectedClass:"selected", // the class used to show a slideButton is active
endOfLoopClass:"endOfLoop", // the class used to show the end of a loop (added to next and previous buttons if looping is disabled and your at the start or end of the list)
trans:'fade', // what type of transition should i use 'fade' or 'slide'
transSpeed:500, // how fast the transition takes place
grouped:true, // if im acting on a slide transition, should I fit as many into the visible are as possible?
perfectFit:true, // sometimes the 'slide' transition panel has an odd size, should I always make it a perfect fit?
externalAction:function(){} // an action that fires after a slide loads passes back the index of the slide loaded

Sample 1: the most basic

JQuery code: $("#slideShow1").slidepanel();
CSS code: #slideShow1{height:500px;}

Sample 2: now with slideButtons

JQuery code: $("#slideShow2").slidepanel({slideButtons:'#panelButtons'});
CSS code: #slideShow2{height:500px;}
.button{height:15px;width:15px;background:url('imgs/button.png');margin:2px;float:left;}
.selected{background:url('imgs/button_selected.png');}
.clearboth{clear:both;}
HTML code: < div id='panelButtons'></div>< div class='clearboth'></div>

Sample 3 : showing off

this example uses irregular sized panels, and calculates which panel to slide to next. It also makes sure that the slides finish with a perfect fit.
JQuery code: $("#alphaPanel").slidepanel({pause:2,trans:'slide'});
CSS code: #alphaPanel{height:20px;width:200px}
